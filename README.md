## Project Description
A C++ console application for automated exam seating arrangement management with teacher and student login systems.

## Features
- Secure teacher PIN authentication
- Student registration and management
- Department-wise and random seat allocation
- Search functionality
- Configurable halls and seats
- Student login with ID verification

## How to Compile and Run

g++ main.cpp -o seat_allocator
./seat_allocator


## Default Credentials
- Teacher PIN: `654321`
- Student ID format: `etsXXXXXX` (6 digits)

## Usage Instructions
1. Teacher login using PIN
2. Register students
3. Configure halls and seats
4. Allocate seats using allocation algorithms
5. Students can login to view their seat assignments
