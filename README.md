## Project Description
The **Exam Hall Seat Allocation System** is a C++ console-based application developed to automate the management of student exam seating arrangements.  
The system supports **teacher and student login**, student registration, and efficient seat allocation across multiple exam halls.

This project is intended for **academic purposes** and focuses on fundamental C++ programming concepts.

---

## Features
- Secure teacher login using a 6-digit PIN  
- Student registration, update, deletion, and search  
- Student login using validated ID format  
- Department-wise (sequential) seat allocation  
- Random seat allocation without conflicts  
- Configurable number of halls and seats  
- View allocated hall and seat information  

---

## How to Compile and Run

g++ main.cpp -o seat_allocator
./seat_allocator

## Usage Instructions

1. Login as a teacher using the PIN
2. Register students and manage student records
3. Configure the number of exam halls and seats
4. Allocate seats using either department-wise or random allocation
5. Students can log in using their ID to view seat assignments
